# SCTEC - Desafio Prático de Software: Agente IA DevOps

## Desafio Escolhido
Desafio Prático de **Software** - Edital SCTEC IA para DEVs (Onda 1)

## 🎯 Objetivo
Agente de IA generativa que **automatiza criação de manifests Kubernetes** via prompts naturais. Integra LLM para produtividade DevOps (K3s, Helm-style).

## 🚀 Como Executar
1. Clone: `git clone https://github.com/SEU_USERNAME/sctec-ia-devops-agent`
2. `pip install -r requirements.txt`
3. Crie `.env` com `OPENAI_API_KEY=sk-...`
4. `python src/agent.py`
